### Command:
#### npm init -y
#### npm i -D babel-cli babel-preset-env
#### npm i -D webpack webpack-cli
#### npm install -D babel-loader@7 babel-core babel-preset-env webpack
#### npm i -D babel-polyfill webpack-dev-server

#### npm run dev
#### npm start
